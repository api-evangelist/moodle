# Moodle GraphQL API

Moodle is an open-source learning management system. The API covers courses, users, enrollments, grades, assignments, quizzes, forums, resources, activities, calendar events, badges, and completion tracking.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Moodle REST API
**Documentation:** https://moodledev.io/docs/apis/subsystems/external/

- Reference: https://moodledev.io/docs/apis/subsystems/external/functions
- Schema: https://github.com/api-evangelist/moodle
